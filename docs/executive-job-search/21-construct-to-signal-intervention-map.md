# Construct-to-Signal and Intervention Map

Updated: 2026-05-31
Item covered: 8

## Mapping Table

1. Self-regulation.
Signals: weekly cadence adherence, missed follow-up ratio.
Interventions: search rhythm contract, rejection recovery protocol.

2. Networking behavior.
Signals: outreach activation count, sponsor-map depth, response rate.
Interventions: network activation burst, board sponsor map.

3. Job-search intensity.
Signals: active pipeline count, actions per week, stage transitions.
Interventions: channel mix review, opportunity-discipline reset.

4. Identity threat.
Signals: repeated narrative edits, confidence decline, delayed outreach.
Interventions: role reframing sprint, proof-point compression.

5. Career adaptability.
Signals: pivot response time, intervention completion, decision lag.
Interventions: timing and context check, first-90-day design.

## Product Wiring Guidance

1. Signals should be observable in product logs.
2. Every intervention should have a success metric.
3. Low-confidence signals should degrade to cautious recommendations.
