# Job Description Language Analysis Method

Updated: 2026-05-31
Item covered: 15

## Objective

Analyze hiring language by role and sub-role to identify recurring requirements, risk cues, and mandate language.

## Method

1. Collect role-relevant job descriptions from public listings.
2. Normalize title families and sub-role tags.
3. Extract recurring language around mandate, scope, constraints, and stakeholder expectations.
4. Build a lexicon of repeated phrases by role segment.

## Weekly Output

1. Role-language lexicon update.
2. New or emerging hiring cues.
3. Delta report against prior week.
