# Public Transition Mining Method

Updated: 2026-05-31
Item covered: 14

## Objective

Mine publicly visible executive transitions for role shifts, function moves, tenure patterns, and timing context.

## Collection Rules

1. Use publicly available profile and role-history information.
2. Store role, function, level, tenure, and move type.
3. Record source URL and extraction timestamp.
4. Do not store private or non-public personal data.

## Weekly Output

1. Transition move patterns by role segment.
2. Tenure-to-transition distributions.
3. Function-shift frequencies.
4. Geography and scope change notes.
