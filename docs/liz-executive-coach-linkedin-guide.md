# Liz Daily Guide: Executive Coach Outreach on LinkedIn

Purpose: Give Liz a repeatable daily operating system for finding, qualifying, and messaging executive coaches on LinkedIn, aligned to the same rhythm used on the Social desk.

Primary references:
- Personas: `docs/content/executive-coaching-synthetic-council.md`
- Objections and replies: `docs/executive-coach-outreach-messages.md` and `docs/outreach-playbook.md`
- Daily operator workflow style: `src/app/(dashboard)/dashboard/admin/social/social-client.tsx`

---

## 1) Daily Operating Rhythm (45-60 minutes)

### Step 1: Open the same core workspace every day (5 min)
- Open LinkedIn feed, messaging, and My Network.
- Open LinkedIn search in a second tab.
- Open the Social admin page and copy the day\'s approved coach connection template style.
- Open your outreach tracker.

### Step 2: Build today\'s target list (15-20 min)
- Find 8-12 new coaches that fit the criteria in Section 2 (LinkedIn-only mode).
- Add each person to the tracker before sending anything.
- Assign a persona tag for each prospect (Transition, VP-to-CXO, Search Affiliate, Board).

### Step 3: Send connection requests (15 min)
- Send 8-12 requests.
- Use one of the connection-note options from Section 4.
- Personalize the first line with one real signal from profile or recent content.

### Step 4: Process follow-ups due today (10-15 min)
- Follow the 7-touch sequence in Section 5.
- Only send touches due today.
- Stop sequence immediately if they reply "not interested".

### Step 5: Log outcomes and objections (5 min)
- Update tracker status for each person touched.
- Log objections using the labels in Section 6.
- Flag any new objection not in the current list.

## 2) How to Find Executive Coaches Who Fit Our Criteria

Use LinkedIn native search first. This guide assumes no Sales Navigator.

### LinkedIn-only search method (repeat daily)
- In LinkedIn search, run: `"executive coach" AND (CIO OR CTO OR "career transition" OR "board")`.
- Filter by `People`.
- Set `Connections` to 2nd-degree first.
- Set `Locations` to United States.
- Set `Current company` only when needed (for boutique firms you already know).
- Open 20-30 profiles, then keep only A/B-quality matches.

### If LinkedIn filters are limited in your account
- Use Google X-ray search and then open profiles in LinkedIn:
- `site:linkedin.com/in "executive coach" "career transition" "CIO"`
- `site:linkedin.com/in "executive coach" "VP to C-level"`
- `site:linkedin.com/in "executive coach" "board" "governance"`
- Add matches to tracker and qualify using persona rules below.

### Core filter set
- Geography: United States (then expand to Canada/UK once US daily volume is stable).
- Relationship: 2nd-degree first.
- Industry: Professional Training and Coaching, Management Consulting, Human Resources Services (if visible).
- Company headcount: 1-10 preferred (can include 11-50 if niche fit is strong).
- Posted on LinkedIn: active in recent content (prefer last 30 days).
- Seniority: Owner, Founder, Partner, Principal, Managing Director.

### Keywords to include
- "executive coach"
- "leadership coach"
- "career transition coach"
- "CIO coach"
- "VP to C-level"
- "board coach"
- "interview prep"

### Exclude terms
- "life coach"
- "wellness coach"
- "mindset coach" (unless executive transition evidence is clear)
- "fitness"
- "relationship coach"

### Persona tagging rules (required)
Based on `docs/content/executive-coaching-synthetic-council.md`:
- Career Transition Specialist: talks about layoffs, transition, landing quickly, search discipline.
- VP-to-CXO Positioning Coach: talks about next-seat narrative, board-ready story, executive branding.
- Search Affiliate Coach: references retained search, finalist prep, offer negotiation around search processes.
- Board/Governance Coach: references board seats, governance pathways, director relationships.

### Qualification score (A/B/C)
- A: Exact persona match + active posting + 2nd-degree + clear transition/search language.
- B: Strong persona match but weaker signal (older posts or broader positioning).
- C: Potential fit but unclear ICP; keep for later.

Send only A and B today.

### Daily volume target without Sales Navigator
- New profiles reviewed: 20-30
- New qualified prospects added: 8-12
- Connection requests sent: 8-12

Quality beats volume. Keep acceptance and reply rates healthy instead of forcing larger daily send counts.

## 4) LinkedIn Message Options Liz Can Use

All options below are built to stay within LinkedIn connection-note limits. Personalize bracketed fields.

### Connection request option 1 (General)

### Connection request option 2 (Career Transition)
[First Name] - noticed your transition coaching work. We built Starting Monday to reduce context rebuild between sessions so coaches can stay strategy-first. Open to connect?

### Connection request option 3 (VP-to-CXO)
[First Name] - your VP-to-CXO perspective stood out. We built Starting Monday so coaches can keep senior candidates moving between sessions with less admin overhead. Open to connecting?

### Connection request option 4 (Search Affiliate)
[First Name] - saw your interview prep/search content. We built Starting Monday to help coaches keep candidates prep-ready between sessions. Happy to connect?

### Connection request option 5 (Board/Governance)
[First Name] - noticed your board advisory focus. We built Starting Monday to help coaches track long-cycle client momentum and prep in one place. Open to connect?

### Post-accept message option A
Thanks for connecting, [First Name].

Most coaches tell us the same issue: too much paid session time goes to rebuilding context. Starting Monday helps you easily stay on top of each client in one place between sessions - signals, prep briefs, and accountability.

Worth a quick 15-minute look?

### Post-accept message option B (lower-friction)
Thanks for connecting, [First Name]. If easier than a call, I can send a 60-second walkthrough and two sample prep briefs so you can judge fit quickly.

### Day-3 no-reply nudge
Quick follow-up in case this got buried. Would a 60-second screen recording be easier than a call?

### Day-7 proof nudge
Happy to send two sample prep briefs so you can see whether this would help you easily stay on top of each client in one place without adding work.

### Day-30 close-the-loop
No pressure if timing is not right. If useful later, reply "2 clients" and I\'ll send the 30-day test setup.

## 5) Concrete 7-Touch Sequence Liz Should Run

- Touch 1 (Day 0): LinkedIn connection request with personalized note.
- Touch 2 (Day 1): Post-accept LinkedIn message or email first-touch.
- Touch 3 (Day 3): LinkedIn no-reply check-in with micro-CTA (video offer).
- Touch 4 (Day 7): Email or LinkedIn proof sample (two briefs).
- Touch 5 (Day 14): LinkedIn value-add insight tied to their persona.
- Touch 6 (Day 21): Email offering 30-day test with two clients.
- Touch 7 (Day 30): Final close-the-loop message with easy re-open keyword.

Rules:
- Each touch must add new value (new proof, new angle, or easier CTA).
- Do not repeat the same ask in the same channel two touches in a row.
- Do not send text/SMS unless there is explicit permission.

- If still unresolved, offer a short call or close the loop.
- Log exact wording in notes for message refinement.

## 8) How Much More Can We Automate?

Short answer: about 60-70% of the workflow can be automated safely; 30-40% should stay human.

### Can automate now (low risk)
- Daily task queue generation (who is due for which touch).
- Persona suggestion from profile keywords.
- Message draft prefill by persona + touch step.
- Follow-up reminders and "next touch" date setting.
- Objection classification from incoming replies.
- Weekly metrics rollups (acceptance, response, demo-booked, objection mix).

Estimated time reduction: 35-45%.

### Can automate with light build work (medium risk)
- CRM-style sequence engine inside admin (step progression + pause rules).
- Draft variant A/B recommendation based on prior response data.
- Auto-create call prep brief when reply intent is positive.

Estimated additional time reduction: 15-20%.

### Should remain human (high judgment / platform-risk)
- Final personalization sentence in connection request.
- Final send decision for each touch.
- Handling nuanced objections and negotiation language.
- Any LinkedIn action that could violate platform anti-automation policies.

### Practical ceiling
- Safe automation ceiling for this motion: 60-70%.
- Fully automated cold outreach is not recommended for LinkedIn trust, deliverability, or account safety.
