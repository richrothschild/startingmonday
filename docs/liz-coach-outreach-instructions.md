## Coach Outreach: The Process

Rich is executing the coach outreach himself, but this is where you can help if needed:

### Step 1: Build the list (Rich does this)
- Sales Navigator search with specific filters (executive coach, 1-10 employees, US, active on LinkedIn, posted in last 30 days)
- Target 10-15 coaches per day
- Create a "Coach Outreach" list in Sales Navigator

### Step 2: Send connection requests (Rich does this)
- Cold connection request with a personalized note (not templated)
- Reference something specific from their profile or a recent post
- Keep under 250 characters

### Step 3: Track everything (Rich does this, but you can audit)
- Google Sheet with: name, LinkedIn URL, request date, status, response date, response type, notes
- This is the single source of truth for follow-ups

### Step 4: Follow the sequence
- **Day 0:** Connection accepted → send follow-up message immediately (same day if possible)
- **Day 3:** No response → send demo video offer
- **Day 7:** Still no response → final friendly touch, then stop

### Step 5: Demo and close (Rich does this)
- Schedule 15-minute walkthrough if they respond positively
- Show how their clients would use it
- Link to `/for-coaches` page if they want to read more
- Move to trial sign-up or "not ready"

## Your Role: What to Watch For

While Rich is doing the outreach, here's what to monitor:

**1. Quality of responses**
- If response rate drops below 15% after the first 30 contacts, the messaging is wrong
- Save any strong responses or objections and flag them for message refinement

**2. Partnership infrastructure**
- Coaches need to feel supported. The `/partners` dashboard is where they track commission
- Make sure the dashboard loads correctly and calculations are accurate
- If coaches complain about commission calculations, escalate to Rich immediately

**3. Supply-side help**
- Coach coaches are often on speaking circuits. When you see them pop up in the speakers database, flag them for Rich
- If you see a coach post about "my clients in transition" or "executive search", that's a prime prospect signal

**4. Referral tracking**
- When a coach signs up through their referral link, confirm they're added to the `partners` table with the right commission rate (20%)
- Audit one week a month: pick 5 random coaches, verify their referral link is working, and check their subscriber count on the dashboard

## Objection-Handling Appendix (5 Common Pushbacks)

Use these when coaches respond with hesitation. Keep replies short and specific.

### 1) "My clients already pay me for this."
- **Reply:** "Exactly - this supports your delivery, it does not replace it. It handles tracking and daily intelligence between sessions so your paid time stays strategic."

### 2) "I do not want another tool to manage."
- **Reply:** "The workflow is simple: Monday pipeline review, every morning signal action, prep brief before interviews. Setup once, then cadence runs daily."

### 3) "My clients are confidential."
- **Reply:** "Private by default. No recruiter visibility, no employer visibility, and client-controlled access. Confidentiality is a core requirement, not an afterthought."

### 4) "AI content is usually generic."
- **Reply:** "The output is company-specific and role-specific. Coaches review and edit before client use, so quality stays aligned with your standard."

### 5) "Not sure the economics are worth it."
- **Reply:** "Partner model is 20% recurring on active referrals. More importantly, clients stay engaged between sessions, which improves both outcomes and retention."

## If Coaches Reach Out

If a coach messages Rich directly or emails us about the partner program:

1. **They already connected:** Move them to demo immediately. This is warm, not cold.
2. **Referral link question:** Make sure they applied through `/partners#apply` and have their referral code
3. **Commission calculation question:** Verify in the partner dashboard. 20% of subscription tier = their commission
4. **Technical issue:** Check if their dashboard is loading. If not, create a fresh login link
5. **Feature request:** Log it and let Rich know. Common asks: team member access, cohort bulk pricing, candidate activation tracking
