# Coach Outreach Pain-Led Messaging System

Purpose: operationalize the coaches-council pain framework into deployable outreach assets.

Core brand wedge:
- Clients arrive underprepared.
- Between-session follow-through collapses.
- Too much time is wasted rebuilding context.

Positioning line:
Starting Monday helps executive coaches stop losing time to underprepared clients, collapsing follow-through, and constant context rebuild.

- Starting Monday app = recurring system solved across clients and weeks.

## Reusable copy pattern

1. Name the pain in plain language.
2. Show the cost of leaving it alone.
3. Offer immediate relief.
4. Clarify micro product solves the moment.
5. Clarify Starting Monday solves the recurring system behind the moment.
