# Icon and Tagline Synthetic Council Review

## Scope

This review evaluates three existing icon assets:

- `starting-monday-logo-option-a.svg`
- `starting-monday-logo-option-b.svg`
- `starting-monday-icon-option-c.svg`

Council members applied:

- Marty Neumeier
- Chris Do
- Brian Balfour
- Andrew Chen
- David Ogilvy
- Rory Sutherland

Each round includes icon and tagline feedback. The council iterated five times before the final recommendation.

## Round 1: Baseline Read

### Option A

- Neumeier: Strong system feel, but too detailed for avatar contexts.
- Do: Premium enough for website hero, weak for small social mark.
- Balfour: Better for trust conversion than top-of-feed recognition.
- Chen: No immediate network cue in small circles.
- Ogilvy: Good promise support when paired with strong headline.
- Sutherland: Sunrise cue is memorable, but not yet sticky at tiny size.

Tagline reaction:

- Current line "Executive Search Operating System" is clear but category-heavy.
- Lacks status and emotional pull.

### Option B

- Neumeier: Clearer now with "SM" monogram, stronger differentiation.
- Do: Better premium signal for profile image usage.
- Balfour: Best chance to build repeat recognition loops in feed.
- Chen: Better atomic-network identity in comments and reposts.
- Ogilvy: Most likely to be remembered after one exposure.
- Sutherland: Feels like a symbol of disciplined insiders.

Tagline reaction:

- Needs less functional wording and more strategic consequence.

### Option C

- Neumeier: Elegant, but semantically abstract.
- Do: Looks product-icon level, less brand-owner authority.
- Balfour: Good app icon, weaker page identity anchor.
- Chen: Acceptable for utility contexts, not for narrative brand memory.
- Ogilvy: Harder to tie to a verbal promise.
- Sutherland: Feels technical, not status-marking.

Tagline reaction:

- Could support operational messaging, weak for prestige positioning.

## Round 2: Core Tensions Identified

1. Small-size recognition versus semantic richness.
2. Functional clarity versus premium aspiration.
3. Category descriptor versus status narrative.

Council alignment:

- Use Option B as primary social/avatar mark.
- Keep Option A as lockup for banner and website contexts.
- Keep Option C as product utility icon only.

Tagline direction shift:

- Move from "what the product is" to "what the user becomes."

## Round 3: First Rewrite Pass

### Icon-specific suggestions

#### Option A adjustments

- Increase symbol block contrast by 8 to 12 percent.
- Reduce left symbol detail by one element in small exports.
- Keep full lockup for web and LinkedIn banner only.

#### Option B adjustments

- Preserve monogram dominance.
- Enlarge letter weight for 48px rendering.
- Keep sunrise arc as differentiator.

#### Option C adjustments

- Reposition as internal product icon only.
- Do not use as company-page avatar.

### Tagline candidate set v1

1. Be ready before the market knows.
2. The executive edge starts early.
3. Run the search before it turns reactive.
4. Early signal. Better outcomes.

Council notes:

- Neumeier and Ogilvy favored 1.
- Sutherland favored 3.
- Do favored 2 for premium tone.
- Balfour and Chen favored 4 for feed readability.

## Round 4: Stress Test by Surface

### Surface fit

- Website hero and banner: Option A + longer strategic line.
- LinkedIn company avatar: Option B only.
- Favicon and compact app context: Option C.

### Tagline by context

#### Long form (banner or hero)

- Be ready before the market knows.

#### Mid form (company page headline area)

- Run the search before it turns reactive.

#### Short form (post graphics)

- Early signal. Better outcomes.

Council concern flagged:

- "Executive Search Operating System" should remain as supporting descriptor, not lead line.

## Round 5: Final Consolidated Recommendation

## Final Icon System

### Primary brand icon (social identity)

- Option B
- Reason: Best blend of immediate recognition, brand initials clarity, and premium signal at small sizes.

### Primary lockup (web and banner)

- Option A
- Reason: Stronger storytelling when space allows text and symbol together.

### Utility icon

- Option C
- Reason: Best for app/favicon contexts, not core brand narrative.

## Final Tagline Stack

### Tier 1 (core promise)

- Be ready before the market knows.

### Tier 2 (operational consequence)

- Run the search before it turns reactive.

### Tier 3 (compact line)

- Early signal. Better outcomes.

### Supporting descriptor

- Executive Search Operating System

## Final Council Grade

- Option A + Tier 1 line: A-
- Option B + Tier 1 line: A
- Option C + Tier 3 line: B+

Best overall system grade:

- A (Option B avatar + Option A banner lockup + Tiered tagline stack)

## Immediate Execution Order

1. Set company avatar to Option B.
2. Set company banner to Option A with Tier 1 line.
3. Use Tier 2 line in About headline copy.
4. Use Tier 3 line on social graphics and short creatives.
5. Keep "Executive Search Operating System" as secondary descriptor in profile and site footer contexts.
