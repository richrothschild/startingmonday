# Starting Monday Icon & UI Style Guide

## Brand Iconography Principles
- **Shape:** Rounded rectangles, soft corners, and geometric simplicity
- **Color:** Deep navy backgrounds (#0F172A, #111827), gradients from orange (#F97316) to peach (#FDBA74)
- **Line:** Bold, clean strokes; minimal detail; no outlines unless for contrast
- **Accent:** Use orange gradients for highlights, dots, or motion lines
- **Typography:** Segoe UI, Arial, or system sans-serif; bold for headings, light for captions
- **Icon Size:** 24x24, 32x32, 48x48 for UI; 120x120+ for hero/marketing
- **Consistency:** All icons must use the same color palette and stroke width as the logo
- **Background:** Prefer solid or subtle gradient backgrounds; avoid transparency on dark UI

## Logo/Icon Usage
- Use only assets from `/public/brand/starting-monday-logo-option-*.svg` or `/public/brand/icon-exploration-v1/` for all UI and marketing
- Never use stock icons or mismatched color schemes
- All new icons must be submitted as SVG and reviewed for color, stroke, and shape consistency

## Example Palette
- Navy: #0F172A
- Slate: #334155, #111827
- Orange: #F97316
- Peach: #FDBA74
- Light Gray: #E2E8F0

## Fortune 100 Quality Checklist
- All icons are custom, not stock
- SVG only, no PNG/JPG
- Consistent stroke width and corner radius
- Color contrast meets WCAG AA
- All icons reviewed quarterly for consistency
