## Mark-Lens Scorecard (Exact Line Edits Per Issue)

Format notes:

- Exact line edit is represented as old line -> new line.
- Each issue also received one new line: top coach pain insight.

### Issue 1

- One action: Implement a 20-minute end-of-day pipeline check with one mandatory next action logged. -> Run a 20-minute end-of-day pipeline check Monday through Friday, and start each session by reviewing last week's completion rate (discipline score = completed checks / 5).
- Added line: Top coach pain insight: Coaches lose strategic session time when week-one updates are reconstructed from memory instead of tracked daily.

### Issue 2

- One action: Create one narrative spine with strategic context, measurable outcome, and repeatable leadership pattern. -> Create one narrative spine, rehearse it in 3 real conversations this week, and review a weekly consistency score (aligned introductions / total introductions).
- Added line: Top coach pain insight: Coaches report repeated rework when clients cannot deliver one stable leadership narrative across conversations.

### Issue 3

- One action: Score each live opportunity on five fixed criteria before prep calls. -> Score each live opportunity on five fixed criteria before every prep call and track weekly decision-discipline score (scored opportunities / active opportunities).
- Added line: Top coach pain insight: Coaches struggle most when clients chase every opportunity and arrive at prep calls without clear tradeoff criteria.

### Issue 4

- One action: Add a monthly governance cadence with three fixed actions. -> Run a monthly governance cadence with three fixed actions and track execution reliability (completed monthly actions / 3).
- Added line: Top coach pain insight: Coaches often lose long-term momentum when clients stop structured positioning habits immediately after placement.

### Issue 5

- One action: Rank top 10 relationships by active-advocacy likelihood and assign next steps. -> Rank top 10 relationships by active-advocacy likelihood, execute the top 5 next steps this week, and track advocacy activation rate (completed touches / 5).
- Added line: Top coach pain insight: Coaches repeatedly see high-contact clients stall because very few relationships convert into real advocacy.

### Issue 6

- One action: Adopt a fixed 4-day final-round protocol. -> Adopt a fixed 4-day final-round protocol and track protocol adherence this week (completed prep stages / 4).
- Added line: Top coach pain insight: Coaches see avoidable final-round misses when prep starts too late and lacks a repeatable rhythm.

### Issue 7

- One action: Launch a five-metric weekly scorecard and review it in session minute one. -> Launch a five-metric weekly scorecard, review it in session minute one, and track scoreboard compliance (weeks reviewed on time / total weeks).
- Added line: Top coach pain insight: Coaches cannot hold execution accountability without a simple shared scoreboard reviewed every session.

### Issue 8

- One action: Schedule monthly board-readiness blocks with three fixed actions. -> Schedule monthly board-readiness blocks with three fixed actions and track continuity score (months completed in sequence / months planned).
- Added line: Top coach pain insight: Coaches lose board-path momentum when governance outreach is treated as ad hoc instead of monthly discipline.

### Issue 9

- One action: Pre-build readiness packs for top five targets. -> Pre-build readiness packs for top five targets this week and track readiness coverage (packs complete / 5).
- Added line: Top coach pain insight: Coaches see qualified clients miss roles because readiness materials are not pre-built before opportunities surface.

### Issue 10

- One action: Remap top three strategic tasks to peak-energy windows. -> Remap top three strategic tasks to peak-energy windows and track execution rate (tasks completed in planned windows / 3 per week).
- Added line: Top coach pain insight: Coaches report that clients miss strategic tasks when important work is scheduled outside peak-energy windows.

### Issue 11

- One action: Add a pre-negotiation worksheet for value case, mandate clarity, and non-negotiables. -> Add a pre-negotiation worksheet for value case, mandate clarity, and non-negotiables, and track prep completeness (fields completed / total fields).
- Added line: Top coach pain insight: Coaches see compensation talks derail when identity anxiety is unaddressed before mandate and value are clarified.

### Issue 12

- One action: Implement a 90-day post-placement cadence with weekly indicators. -> Implement a 90-day post-placement cadence with weekly indicators and track checkpoint reliability (completed check-ins / planned check-ins).
- Added line: Top coach pain insight: Coaches often watch first-90-day success drift without a lightweight weekly checkpoint system.
