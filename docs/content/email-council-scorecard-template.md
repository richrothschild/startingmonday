## Hard-Fail Check (Auto-Block)

Mark pass or fail for each:

- Trigger appears in first 2 lines: pass | fail
- Any outcome claim includes denominator and caveat: pass | fail
- CTA is explicit yes or pass: pass | fail
- No pressure language that removes safe-to-say-no posture: pass | fail
- Persona specificity (cannot be reused unchanged across unrelated personas): pass | fail
- No implied guarantee language: pass | fail

If any fail, status is blocked until corrected.
