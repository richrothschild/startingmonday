# AP Style Compliance Policy

All content (outreach messages, guides, public-facing docs, and major UI copy) must be reviewed using the AP Style Compliance Checklist (see docs/ap-style-compliance-checklist.md) before completion and deployment.

## Policy Steps
1. Draft content in markdown or the appropriate format.
2. Review the draft using the AP Style Compliance Checklist.
3. Mark the checklist as completed (digitally or in a PR comment).
4. For high-visibility or external content, do a manual pass referencing docs/ap-style-quick-reference.md and, if needed, the full AP Stylebook.
5. Only mark the content as "ready for review" or "done" after checklist completion is confirmed.

## Enforcement
- PR reviewers must confirm checklist completion before approving merges.
- For docs and outreach, include a note in the PR or commit message: "AP style checklist completed."
- For UI copy, reference the checklist in the design or QA review.

---

This policy ensures consistent, professional, and AP-compliant communication across all channels.
