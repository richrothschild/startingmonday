## Core features

### Pre-Session Snapshot
Before each client session, review one screen that summarizes what changed since the last conversation: new signals, pipeline movement, confidence trend, commitments kept or missed, and one recommended talking point for the upcoming session. In the Jan-May 2026 pilot cohort, coaches reported materially less context-rebuild time; results vary by workflow and adoption quality.

### Client Portfolio Dashboard
A portfolio-level view of all active clients: active vs drifting status, confidence trend, pipeline velocity, prep quality score, and last session date. It is designed to highlight which clients need intervention first.

### Client Visibility Controls (client-controlled)
Clients set what coaches can see. Options include read-only signals, pipeline access, full prep brief access, or no access. Coaches cannot view anything that has not been granted.

### Session Commitment Tracker
Log commitments made in each session. The tracker surfaces follow-through, at-risk commitments, and overdue actions before the next call.

### Coach Partner Program
Earn 20% recurring commission on active client referrals. Enrollment is free at startingmonday.app/partners and includes a referral link, onboarding materials, and messaging templates.

### Practice Model Paths
Choose the coach persona route that matches how you operate:
- Independent coach (solo practice): 1-15 clients, lightweight portfolio view, referral-only commission model
- Boutique firm coach: Team-based operations, shared client visibility settings, standardized scorecards
- Enterprise-sponsored coach cohort: Cohort governance, sponsor-ready reporting, program-level risk queue
- Search-affiliate coach: search-adjacent workflow with trust-boundary framing and interview-arc support

### 30-Day Pilot Scorecard
A structured pass/fail pilot framework for evaluating the platform with 2-3 clients. Metrics include context-rebuild time, prep quality, between-session momentum, and client confidence trend.

### Weekly Activity Digest
Each week, receive a portfolio summary: who is active, who is drifting, newly surfaced signals, and the client most at risk of losing momentum.

### Trust Pack
Documentation for coaches and firms evaluating broader rollout: methodology notes, claims policy, pilot evidence with denominator disclosures, security posture, and diligence path. See startingmonday.app/for-coaches/trust-pack.

### Boutique Firm Tools (Boutique plan and above)
- Shared client views across the coaching team
- Standardized session scorecard template configurable per firm
- Coach performance benchmarking (aggregate and anonymized)
- Sponsor-ready cohort reporting for enterprise-sponsored programs

### Search-Affiliate Workflow
For coaches working alongside retained search firms: trust-boundary documentation, interview-arc visibility, and proof assets for partner communication.

## What coaches see vs. what clients control

| Feature | Client controls | Coach sees |
|---------|-----------------|------------|
| Target company list | Client adds/removes | Visible if client grants |
| Signal feed | Client's signals only | Read-only if granted |
| Pipeline status | Client manages | Summary view if granted |
| Prep briefs | Client generates | If client shares |
| Confidence scores | Auto-generated | Always visible to coach |
| Commitment tracker | Set in session | Visible to coach |

## Session workflow

| Touchpoint | What to do | Time required |
|-----------|------------|---------------|
| Day before session | Open pre-session snapshot | 5 min |
| Start of session | Reference what-changed summary | 2 min |
| During session | Log commitments in real time | 1-2 min |
| After session | Update next-action and risk markers | 2 min |
| Weekly | Review portfolio risk queue | 10 min |
