# Hero Dual-Track Council Rubric

## Purpose

Prevent top-fold cognitive-load regressions on marketing hero sections while preserving conversion performance.

## Board Model

- Track A: Conversion Council
- Track B: Clarity Council
- Tie-break arbitration: Product + UX

## Synthetic Council (Option 2 Composition)

- Conversion Scientist
- Growth Product Lead
- Lifecycle Strategist
- Analytics Lead
- Information Architect
- UX Psychologist
- Editorial Lead
- Accessibility Specialist
- Frontend Performance Engineer

## Rubric Policy (Required)

- Conversion pass score must be >= 90
- Clarity pass score must be >= 90
- Any individual metric score below 80 is a hard stop

## Conversion Track Metrics

- Value proposition focus
- CTA discipline
- Trust signal balance
- Proof specificity
- Scan efficiency

## Clarity Track Metrics

- Copy brevity
- Hierarchy simplicity
- Information chunking
- Reading flow
- Decision path clarity

## Weekly Operating Cadence

- Run hero dual-track audit weekly and on demand
- Post summary to Slack for each run
- Send summary email for each run to richard@startingmonday.app
- Store machine-readable output in docs/hero-dual-track-council.latest.json
