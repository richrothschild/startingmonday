# Phase 4 Moderated Session Script

Duration: 30 minutes
Audience: first-session executives

## Moderator Intro

Thank you for helping test the first-session experience.
We are evaluating the product, not you.
Please think out loud while you navigate.

## Scenario

You just created your account.
You want to know exactly what to do next and why it matters.

## Tasks

Task 1: onboarding clarity
- Start onboarding and explain what you think the product will do for you next.
- Success signal: user states role timing and company monitoring outcome in plain language.

Task 2: next-best-action clarity
- On dashboard start, choose what you would do first.
- Success signal: user selects highlighted next action without scanning all tasks for more than 10 seconds.

Task 3: briefing comprehension
- Open daily briefing and describe the single most important action for today.
- Success signal: user identifies one concrete next move.

Task 4: trust and confidence
- Ask: what, if anything, makes you hesitant to rely on this workflow?
- Capture trust blockers verbatim.

## Debrief Questions

1. What felt most clear?
2. What felt repetitive or confusing?
3. If you had 5 minutes tomorrow morning, what would you open first?
4. What should be removed?

## Scoring Prompts

- Time to first confident action (seconds)
- Wrong turns count
- Moderator confidence score (1-5)
- User confidence score (1-5)

## Session Log Format

- Participant ID
- Date and time
- Completion status per task
- Notable quotes
- Top 3 friction points
- Severity suggestion (1-4)
