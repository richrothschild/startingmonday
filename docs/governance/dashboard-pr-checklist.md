# Dashboard PR Checklist

Copy this into PRs that add or modify dashboard pages.

```md
### Dashboard standards compliance
- [ ] Architecture aligns with docs/dashboard-page-architecture-standard.md
- [ ] Copy/tone aligns with docs/dashboard-copy-and-tone-standard.md
- [ ] Instrumentation aligns with docs/dashboard-instrumentation-standard.md
- [ ] First action is clear within 10 seconds (desktop and mobile)
- [ ] Primary + support interactions are tracked with section/action/target properties
- [ ] I used docs/templates/dashboard-page-template.md for implementation structure
```

## Reviewer prompt

Reviewers should request changes if any checklist item is unchecked or unsupported by code/docs evidence.
