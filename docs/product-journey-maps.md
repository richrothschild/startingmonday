## AI Recruiter Platform — Full Behavioral Coverage

---

## How to Read This

Each persona is mapped through 10 stages. For every behavior:
- **What they do** — the specific action the job seeker takes
- **Pain / Friction** — where they get stuck or do it wrong
- **Product feature** — what the platform does to help

Stages are the same across all personas so you can compare.
Behaviors that are identical across personas appear once and are referenced thereafter.

## Feature Legend

| Feature | What It Does |
|---|---|
| **LinkedIn Optimizer** | Analyzes and rewrites LinkedIn headline, About, Experience, Skills |
| **Resume Tailoring** | Rewrites resume bullets for a specific job description |
| **Company Discovery** | Suggests target companies based on profile + market data |

| **AI Chat** | Conversational AI with full search context — advises, drafts, coaches |
| **Outreach Drafting** | Generates cold emails, LinkedIn messages, follow-ups in user's voice |
| **Pipeline Manager** | Tracks every application, contact, stage, and follow-up date |
| **Follow-Up Manager** | Reminders + auto-drafted follow-up messages when contacts go stale |
| **Daily Briefing** | 7am email: 3-5 prioritized actions based on live pipeline state |

| **Positioning Coach** | Multi-turn coaching on how to frame background, pivot, or level jump |
| **Salary Intelligence** | Market comp data + negotiation script for offer stage |
| **Market Intelligence** | Weekly digest of new postings and funding signals in target sectors |
| **Momentum Score** | Weekly velocity score + nudge when search stalls |

| **Resignation Coach** | Resignation letter drafting, conversation prep, counter-offer advisor |
| **90-Day Briefing** | First 90 days plan + relationship-building strategy for new role |

# Persona 1 — The Transformation Executive
*CIO/CTO/VP IT, 20+ years. Highly targeted search. Knows what they want; needs execution.*

---

## Stage 1: Trigger

| Behavior | Pain / Friction | Product Feature |
|---|---|---|
| PE firm buys company, new ownership signals leadership change | Uncertain timeline — do I start now or wait? | **AI Chat** — "Should I start my search now or wait 90 days?" gives context-aware advice |
| Receives separation package, reviews severance terms | Doesn't know if severance is negotiable; whether to sign immediately | **AI Chat** — "Is this severance package standard for my level and tenure?" |
| Decides to make proactive move after being passed over for board seat | Unclear whether to disclose search to current employer | **AI Chat** — stealth vs. open search strategy advice |
| Tells close professional confidants they are looking | Inconsistent message — story isn't tight yet | **Positioning Coach** — lock down the narrative before it spreads |

## Stage 2: Preparation

# Subscription Retention by Stage

The intelligence subscriber (Persona 4) stays subscribed indefinitely because the
**Maintenance** row never goes away. Every other persona churns at the end of
the **First 90 Days** row — but many resubscribe for the next search.

Target the **Trigger** stage for acquisition (LinkedIn ads to recently laid-off,
outplacement firm partnerships, LinkedIn Optimizer as a free trial hook).

Target the **Maintenance** row for retention (Market Intelligence, Momentum Score,
Progress Report create ongoing value between active search periods).
