## What makes this more credible

Use these elements in the final message whenever possible:

1. Add a likely role type, not just a company name.
2. Add a one-line reason the role may open soon.
3. Add an evidence note that points to the kind of public signal used.
4. Mention that the product can surface likely decision-makers and contact information.
5. Keep the workflow description high-level so you do not give away the store.
6. Use the role title language the buyer would actually use, such as VP, Director, Head of, or Chief.

When to name specific signals in the brief:

- Good: one or two high-level signals like leadership change, PE activity, acquisition, expansion, guidance change, or hiring cadence shift.
- Avoid: listing too many signals in the DM itself; it starts to feel like a methodology dump.
- Best practice: put the signal label in the full brief, then keep the DM to the company, likely role, and decision-maker hook.

If you do mention a signal in the DM, use only one phrase, for example:

> leadership change + expansion cadence

or

> PE activity + hiring cadence shift

## Can you add a file to the LinkedIn DM?

Usually, not as a true file attachment in standard LinkedIn direct messages. The safer pattern is:

1. Send the short DM in chat.
2. If you want to include the sample brief, paste a trimmed version in the message or send a link to a PDF/Doc instead.
3. If you want it to feel polished, attach the file in email and keep the DM as the note that tees it up.

If you want, I can also make you a one-line LinkedIn note that introduces the file cleanly, for example:

> I attached a short sample brief so you can see the format I had in mind.

Suggested footnote style:

> Evidence note: public leadership movement, hiring cadence, and adjacent market signals.

If you have the person’s LinkedIn profile URL and email, you can also add a tiny header above the brief:

> Source context: LinkedIn profile + conversation + public market signals.
