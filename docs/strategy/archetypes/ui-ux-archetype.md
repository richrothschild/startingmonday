# UI/UX Archetype

Purpose
- Drive conversion by reducing cognitive load for executives scanning quickly.

Interaction Principles
- Reveal hierarchy in one glance.
- Keep paths obvious and limited.
- Minimize scrolling before first decision.

Layout Rules
- Hero: promise + two CTAs + trust line.
- Middle: three pillars with one-line utility.
- Evidence block: concise, comparative, actionable.
- End: single conversion panel.

Spacing Rules
- Use large rhythm spacing between major sections.
- Tighten internal spacing for related text.
- Never crowd CTA with dense body copy.

Accessibility Rules
- Body text at comfortable line length.
- Strong color contrast for all text and controls.
- Focus states visible.

Scoring Rubric (0-5 each)
- Scanability
- Cognitive load
- CTA clarity
- Mobile ergonomics
- Readability

Pass Threshold
- 22/25 or higher.
