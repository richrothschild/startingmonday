# Anne Applebaum Review of Email Templates

Date: 2026-05-26
Reviewer: Anne Applebaum (Email Council)
Review goal: improve open and reply rates by increasing executive credibility, precision, and seriousness while preserving low-friction yes/pass conversion.

Templates reviewed:

1. Executive initial (engine)
2. Executive follow-up 1 (engine)
3. Executive follow-up 2 (engine)
4. Executive follow-up 3 (engine)
5. Coach initial (engine)
6. Search-firm initial (engine)
7. Outplacement initial (engine)
8. Static coach sequence docs (Day 0/1/3/7/14/21/30)

## Cross-Template Findings

1. Copy quality has improved materially, but several lines still sound like outbound software copy instead of peer-to-peer strategic correspondence.
2. Open rates will improve further if subject lines include a concrete trigger or consequence, not only readiness framing.
3. Credibility is highest where the email names one clear decision and one bounded proof line.
4. Reply intent weakens when phrases are repeated across touches without a new observation.
5. Keep language plain and exact; avoid institutional jargon unless the recipient uses it publicly.

## Template Feedback and Required Changes

### 1) Executive Initial (engine)

Current grade: A-

What works:

- Subject is concise and relevant.
- Decision line is clear and practical.
- Proof line is legal-safe and specific.

What to change for A+:

1. Add one trigger anchor in the first two lines (for example, role shift, market event, or leadership transition signal).
2. Replace stood out phrasing with an explicit reason the note is being sent now.
3. Keep body under 115 words after personalization.

Recommended line upgrade:

- Replace: I have been following your work at [company], and this stood out for [focus] transitions.
- With: I am reaching out because [specific trigger] makes [focus] readiness unusually time-sensitive this quarter.

### 2) Executive Follow-up 1 (engine)

Current grade: B+

What works:

- Shorter than prior version.
- Low-friction ask remains intact.

What to change for A+:

1. Add one new observation not present in initial.
2. Remove generic quick follow-up opener.
3. Add one time boundary to increase response clarity.

Recommended line upgrade:

- Replace: Quick follow-up on my note about [focus] readiness at [company].
- With: Following up because this week is the right window to pressure-test [focus] messaging before broader outreach.

### 3) Executive Follow-up 2 (engine)

Current grade: A-

What works:

- Most concrete asset framing in sequence.
- Good benchmark specificity.

What to change for A+:

1. Keep this as the highest-information touch, but add one calibrated question to surface hidden resistance.
2. Remove any phrasing that sounds like template rotation.

Recommended line upgrade:

- Add before CTA: Would it be unreasonable to test this against one live conversation before deciding?

### 4) Executive Follow-up 3 (engine)

Current grade: A-

What works:

- Strong close-loop tone.
- Body length is appropriate for final touch.

What to change for A+:

1. Add explicit archive behavior to reassure the recipient that attention cost ends.
2. Keep proof sentence only if it adds new information; otherwise remove in final touch.

Recommended line upgrade:

- Replace CTA line with: Reply yes and I will send the benchmark. Reply pass and I will archive this thread.

### 5) Coach Initial (engine)

Current grade: A-

What works:

- Good friction language around context rebuild.

What to change for A+:

1. Include one concrete coach-visible outcome that sounds like session quality, not software process.
2. Keep one asset ask and remove any extra framing language that can read like methodology branding.

Recommended line upgrade:

- Add after decision line: The practical gain is more paid session time spent in judgment, not recap.

### 6) Search-Firm Initial (engine)

### 7) Outplacement Initial (engine)

Recommended line upgrade:

### 8) Static Coach Sequence Docs (Day 0/1/3/7/14/21/30)

Current grade: A-

What works:

- Yes/pass logic is largely consistent.

What to change for A+:

1. Day 14 still contains platform-forward language; reframe to coach outcome first.
2. Day 21 should use one narrower success metric in the body.
3. Day 30 should include explicit archive behavior in every variant.

Recommended line upgrades:

- Day 14 replace: That is why we built Starting Monday...

- Day 30 add: If not useful right now, reply pass and I will archive this thread.

## Priority Change Order (Anne Applebaum)

1. Executive follow-up 1 trigger precision and time boundary.
2. Executive initial trigger anchoring.
3. Search-firm opening polish and consequence precision.
4. Day 14 and Day 30 static sequence tightening.
5. Outplacement differentiation sentence.

## Send-Gate Standard from Anne Applebaum

Each template should fail send if any of the following is true:

1. Contains a generic opener without a credible trigger.
2. Uses broad claim language without bounded proof.
3. Uses process-jargon where plain language would increase trust.
4. Asks for response without a clear decision rationale.

Recommended send threshold after fixes: 95 out of 100 council score.
