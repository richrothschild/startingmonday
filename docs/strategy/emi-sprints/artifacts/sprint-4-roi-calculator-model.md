## Objective

Provide a transparent model for evaluating outplacement pilot commercial impact.

## Input Variables

1. Cohort size
2. Program duration
3. Counselor time per participant (baseline and pilot)
4. Completion rate baseline and projected
5. Pilot license cost

## Output Metrics

1. Counselor time saved (hours)
2. Estimated completion lift
3. Cost per completed participant
4. Estimated margin impact
5. Proposal-to-pilot expected conversion influence
