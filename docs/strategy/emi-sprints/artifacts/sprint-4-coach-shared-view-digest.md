# Sprint 4 Artifact: Coach Shared-View Progress Digest

Date: 2026-05-25
Owner: Product
Status: Completed

## Objective

Provide coaches with a weekly shared-view digest that reduces session context rebuild and improves strategy time.

## Digest Structure

1. Weekly momentum summary
- completed actions
- missed actions
- momentum trend

2. Readiness highlights
- role-target clarity trend
- narrative readiness status

3. Relationship movement
- outreach completed
- stale relationships flagged

4. Coach intervention prompts
- one suggested intervention for discipline
- one suggested intervention for narrative

## Delivery Cadence

1. Weekly send to coach dashboard view.
2. Optional pre-session digest export.

## KPI Link

Primary KPI: coach-reported session effectiveness lift.

## Release Notes

1. Shared-view digest template completed for pilot use.
2. Intervention prompts included for session prep quality.
