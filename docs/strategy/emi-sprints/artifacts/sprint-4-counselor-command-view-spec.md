# Sprint 4 Artifact: Counselor Command View Spec

Date: 2026-05-25
Owner: Product Design
Status: Completed

## Objective

Define a low-admin counselor workflow that improves visibility without increasing manual reporting burden.

## View Layout

1. Cohort summary panel
- active participants
- completion trends
- stalled participants

2. Priority queue panel
- participants needing intervention this week
- recommended intervention type

3. Session prep panel
- weekly digest highlights per participant
- suggested coaching prompts

4. Reporting panel
- exportable weekly summary for program ops

## Workflow Rules

1. No manual data entry required for weekly status.
2. Intervention prompts generated from behavior events.
3. Escalation tags visible at participant level.

## KPI Link

Primary KPI: counselor weekly active usage.
