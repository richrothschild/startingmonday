# Sprint 4 Artifact: Outplacement Pilot Runbook

Date: 2026-05-25
Owner: Partnerships
Status: Completed

## Objective

Define a repeatable pilot operating model that can be run without founder dependency.

## Pilot Scope

1. Duration: 30 days
2. Cohort size: 10-25 executives
3. Roles: program ops lead, counselor lead, partner success contact

## Pilot Phases

1. Setup (Week 0)
- kickoff and success criteria lock
- access and workflow onboarding

2. Activation (Week 1)
- assessment completion and mode assignment
- first weekly review

3. Cadence (Weeks 2-3)
- weekly digests for counselors
- cohort performance checkpoints

4. Close (Week 4)
- KPI review and renewal decision

## Ownership Model

1. Program ops lead: delivery governance
2. Counselor lead: cohort coaching workflows
3. Partner success contact: escalation and reporting

## Escalation Path

1. Workflow issue -> partner success contact
2. Data issue -> operations owner
3. Scope or contract issue -> partnerships lead

## KPI Link

Primary KPI: pilot approval rate from qualified opportunities.

## Release Notes

1. Pilot runbook finalized with explicit owner and escalation model.
2. Founder-independent delivery path documented.
