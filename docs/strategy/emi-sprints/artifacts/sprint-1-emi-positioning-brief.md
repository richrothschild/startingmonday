# Sprint 1 Artifact: EMI Positioning Brief

Date: 2026-05-25
Owner: Founder Office
Status: Draft for Sprint 1 execution

## Category Claim

Starting Monday is the Executive Momentum Intelligence platform for senior technology leaders who need to run disciplined career motion before transition pressure forces reactive behavior.

## Best-Fit ICP

Primary:

1. VP and C-level technology executives in the 0-18 month transition-risk window.
2. Executives currently in search who need daily operating cadence.

Secondary:

1. Executive coaches managing transition clients.
2. Outplacement and transition firms running cohort programs.

## Competitive Alternative

Primary alternative:

1. Manual workflow stack (spreadsheets, notes, ad hoc recruiter outreach, inconsistent follow-through).

Secondary alternatives:

1. Generic job boards and generic networking tools.
2. Coaching without between-session execution infrastructure.

## Differentiation

1. Momentum-first operating model, not listing-first discovery.
2. Daily action loop with low-friction recovery mechanics.
3. Evidence-backed benchmark layer with denominator and confidence discipline.
4. Coach and counselor enablement surfaces that improve session quality.

## Value Conditions

EMI works best when:

1. User commits to a weekly cadence.
2. User maintains target-company and relationship inputs.
3. User uses daily loop and weekly review consistently.

EMI underperforms when:

1. User treats the platform as passive research only.
2. User avoids action loops and feedback prompts.

## Message Hierarchy

1. Core promise: run executive transition as an operating system, not an emergency response.
2. Core outcome: measurable momentum, reduced stall cycles, improved readiness.
3. Core proof: behavior and benchmark evidence with explicit denominator and timeframe.

## Positioning Do and Do Not

Do:

1. Use language of readiness, discipline, and operating cadence.
2. Name the alternative explicitly as reactive manual process.
3. Tie claims to measurable behavior or benchmark evidence.

Do not:

1. Frame as a generic job-search helper.
2. Lead with feature list before category and outcome.
3. Make confidence claims without denominator and timeframe.

## Standard One-Liner

Executive Momentum Intelligence for senior operators who want to run disciplined career motion before the market forces them into reactive search.
