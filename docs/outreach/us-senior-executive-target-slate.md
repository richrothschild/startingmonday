# US Senior Executive Target Slate

## Purpose

This is the immediate US executive outreach slate aligned to Starting Monday's strongest-fit and medium-fit personas.

- Primary lanes: CFO, COO, CIO, CHRO, CRO
- Secondary lanes: CTO, CISO, CDO
- Primary size band: 1,001-10,000 employees
- Channel emphasis: search-firm-adjacent plus board and investor referral pathways

---

## What Is Included

- A ranked Top 100 US target-account executive slate in CSV format
- Role-by-role targeting per account
- Priority, fit tier, channels, and outreach angle for each row
- Ready-to-execute status tracking fields

## Important Operating Note

Executive occupants change often. This slate is account-and-role first so your team can move now.

Recommended enrichment workflow before outbound:
1. Confirm current role holder in Sales Navigator / company IR / leadership pages.
2. Add direct contact route (email path, assistant, or referral path).
3. Validate trigger event (funding, M&A, CEO change, restructuring, board transition).
4. Personalize outreach angle using the persona and trigger.

---

## Suggested Execution Cadence

1. Work top 25 accounts first in 2-week sprints.
2. Use CFO/COO/CIO lanes as initial wedge.
3. Add CHRO/CRO lane in sprint 2 after initial messaging learnings.
4. Run CTO/CISO/CDO medium-fit motions as secondary tests.
5. Track conversion by channel: search firm referral, board referral, investor referral, direct.
