## How to Use This Template

The review is not a reporting exercise. It is a diagnostic. Each completed section tells you something about where the search is healthy and where it is not.

If the conversations section is consistently empty, the outreach is not working or not happening. If the outreach section is full but the conversations section is empty, the outreach is landing wrong. If the signals section fires week after week but produces no action, you are monitoring without moving.

Run the review at the same time every week. Monday morning, before the week starts, is the right moment — it sets the priorities before anything else fills the calendar. The review that happens "sometime next week" does not happen.

The weekly review is also the forcing function for logging conversations and follow-ups promptly. A conversation logged the day after it happens is a conversation you can act on. A conversation you try to reconstruct on Monday morning from memory is a conversation where the context is already gone.
