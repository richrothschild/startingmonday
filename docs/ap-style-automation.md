# AP Style Automation and Tooling

## Automated AP Style Checks

There is no official AP Stylebook API for automated checks. However, you can use third-party tools to help enforce AP style in your workflow:

### 1. Grammarly Business
- Offers AP style checks (subscription required).
- Can be used as a browser extension or in Google Docs/Word.
- Enable AP style in settings for outreach and guides.

### 2. ProWritingAid
- Offers AP style suggestions (subscription required).
- Integrates with browsers, Word, and Google Docs.
- Set style to "AP" in the tool for best results.

### 3. Markdown Linting (Partial)
- Use markdownlint for basic style and clarity checks.
- Not AP-specific, but helps with structure and readability.

### 4. Manual Review
- Use docs/ap-style-quick-reference.md and docs/ap-style-compliance-checklist.md for manual review.
- For critical content, do a manual pass referencing your licensed AP Stylebook.

## Workflow Suggestion
1. Draft content in markdown.
2. Run through Grammarly or ProWritingAid with AP style enabled.
3. Review with the AP style checklist.
4. For high-visibility docs, do a final manual pass with the full AP Stylebook.
