# Outplacement Pilot Runbook

## Purpose
Use this runbook to execute a 30-day cohort pilot with explicit pass/fail criteria.

## Week 0
- Confirm cohort roster and role owners.
- Approve scorecard baseline and intervention thresholds.
- Deliver counselor enablement briefing.

## Week 1
- Launch participant kickoff communication.
- Complete setup and baseline target lists.
- Run first operating review.

## Week 2
- Require at least one signal-driven action per active participant.
- Review red-flag queue and assign interventions.
- Confirm prep-brief usage on high-stakes meetings.

## Week 3
- Validate session-yield improvements.
- Review intervention outcomes.
- Draft day-30 decision memo with evidence.

## Day 30 Decision Gate
- Pass: criteria met, define scale plan.
- Tune: partial criteria met, run corrective 30-day cycle.
- Stop: criteria not met, close without expansion.
